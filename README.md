## About

Open Synapsis builds **open source tools for network intelligence and analysis**. Our projects are modular, ML-driven, and designed for cybersecurity applications. The organization is used to gather tools developped during PhD studies.

-----

## Projects

### [NEXUS](https://github.com/opensynapsis/nexus)

Modular network analysis framework.

- Multi-format support (NetFlow, PCAP, **Nxcap**)
- Real-time traffic analysis
- High-performance packet processing
- Packet Deduplocation algorithms
- TUI interface

**Stack:** Python, C, [Go, Rust]
